# Zhai Ltd. 债氏集团

债氏集团总裁试图用此 app 管理公司的所有狗子.

## 依赖

尽量都下个最新的稳定的版本 (*latest stable releases*) 吧 :P

* `flask`: Web 框架
* `jinja2`: 模板引擎
* `werkzeug`: 实用工具
* `pyjwt`: Authentication
* 数据库 *待定*

## 安装/运行

没有部署 (*deploy*) 的需求, 就随便点吧 :P

```bash
$ python3 app.py
```

## 其他
